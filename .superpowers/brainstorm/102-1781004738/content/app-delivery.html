Comment les parents accèdent-ils à l'app ?

Ce choix détermine tout : notifications, distribution, développement, maintenance

A

Application Web Progressive (PWA)

Site web installable sur l'écran d'accueil du téléphone. Notifications push via le navigateur. Aucun App Store. Mise à jour automatique. Fonctionne sur iOS & Android.

Avantages

  • Déployable en semaines
  • Pas de frais App Store
  • Une seule codebase
  • Hébergeable sur NAS

Limites

  • Notifications iOS limitées (iOS 16.4+)
  • Expérience légèrement moins native
B

App Native (React Native / Expo)

Vraie app iOS + Android distribuée via App Store et Google Play. Notifications push complètes (FCM/APNs). Accès caméra natif pour la selfie station.

Avantages

  • Notifications fiables et riches
  • Expérience native optimale
  • Caméra/biométrie accès complet

Limites

  • Comptes développeur ($99/an Apple)
  • Délai de review App Store
  • Plus complexe à maintenir
C

PWA + Notifications par email/SMS

Web app simple, sans installation obligatoire. Les alertes se font par email ou SMS (Twilio/Brevo) plutôt que par push natif. Le plus simple à construire.

Avantages

  • Zéro friction pour le parent
  • Aucune installation requise
  • Déploiement immédiat

Limites

  • Notifications moins immédiates
  • Dépend d'un service externe (SMS $)
